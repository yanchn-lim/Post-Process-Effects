# Post-Process Effects
This is a collection of post-processing effects that I have learnt about and tried to implement into unity's built-in render pipeline.

## Current Features
- Greyscale
- Sharpness
  - Sharpen Convolution
  - Unsharp Mask
- Colour Correction
    - Contrast
    - Brightness
    - Saturation
    - Gamma
